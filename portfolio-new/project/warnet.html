Project Overview

Platform reservasi digital untuk Game Center yang memungkinkan pengguna memesan paket bermain secara mudah dan praktis. Sistem ini mempermudah manajemen operasional warnet serta memberikan kepastian ketersediaan slot PC bagi pelanggan sebelum datang ke lokasi.

My Process

Pengembangan dimulai dengan merancang database relasional untuk menyimpan data paket dan transaksi. Saya menggunakan Laravel untuk membangun logika backend, mulai dari migration database hingga controller pemesanan. Tahap akhir adalah merancang tampilan frontend responsif agar pelanggan dapat memesan paket dengan mudah.

The Challenge

Membangun logika pemesanan yang akurat di mana status transaksi berubah otomatis dari 'pending' ke 'success', serta memastikan keamanan data pada halaman Login dan validasi agar tidak terjadi duplikasi pesanan.
