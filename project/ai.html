Project Overview

Website pintar pendeteksi objek instan yang dilatih menggunakan Google Teachable Machine. Didukung oleh TensorFlow.js, aplikasi ini bekerja sangat cepat serta menjaga privasi pengguna karena seluruh data diproses langsung di perangkat tanpa perlu dikirim ke server luar.

My Role
Frontend & AI Integration
Tools
TensorFlow.js, JavaScript, HTML, CSS, Teachable Machine
Timeline
2 Months

My Process

Melatih model AI menggunakan Google Teachable Machine, lalu mengekspornya ke format TensorFlow.js untuk diintegrasikan ke website agar mampu mendeteksi objek secara real-time langsung di browser pengguna.

The Challenge

Menjaga akurasi deteksi di berbagai kondisi cahaya serta mengoptimalkan performa agar proses inferensi AI berjalan ringan tanpa membebani resource browser pengguna.
