Tambang Quiz

Project Overview

Game kuis interaktif berbasis web yang mengadu dua tim dalam kompetisi tarik tambang digital. Setiap jawaban benar menggerakkan tali ke arah tim tersebut, menciptakan pengalaman belajar yang seru, kompetitif, dan visual bagi para pemain.

My Role
Frontend Developer
Tools
HTML, CSS, JavaScript
Timeline

Screenshots

My Process

Saya merancang mekanik permainan terlebih dahulu — bagaimana tali bergerak sesuai skor tim, kapan game berakhir, dan cara menampilkan pertanyaan secara acak. Setelah logika selesai, saya membangun antarmuka visual yang intuitif dan menarik menggunakan animasi CSS murni.

The Challenge

Membuat animasi tali yang smooth dan responsif terhadap setiap jawaban, serta membangun sistem state permainan yang dapat mengelola giliran tim, skor, dan kondisi menang/kalah secara real-time tanpa backend.
